## Instance Format

An instance with `N` jobs is given by a plain text file with `n+2` lines:
- Line 1: Number of machines `m`  
- Line 2: Number of jobs `n`  
- The following `n` lines specify the job processing times `p_i`

For the instance source, see:
**References**
- P.M. França, M. Gendreau, G. Laporte, and F.M. Müller.  
  *A composite heuristic for the identical parallel machine scheduling problem with minimum makespan objective.*  
  Computers & Operations Research, 21:205–210, 1994.

- A. Frangioni, E. Necciari, and M.G. Scutellà.  
  *A multi-exchange neighborhood for minimum makespan machine scheduling problems.*  
  Journal of Combinatorial Optimization, 8:195–220, 2004.
